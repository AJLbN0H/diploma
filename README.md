# Платформа для самообучения студентов

## Описание
### Платформа обеспечивает возможность авторизации и аутентификации пользователей, предоставляет функционал для создания и управления разделами и материалами, а также тестирования знаний. Реализована система ролей с различными уровнями доступа для администраторов, преподавателей и студентов.

## В проекте использованы

### Фреймворк:
- Django с Django Rest Framework (DRF)
### База данных:
- PostgreSQL
### ORM:
- Встроенный ORM Django
### Документация:
- Swagger
### Удаленный репозиторий:
- Git
### Безопасность:
- CORS.
- JWT токены.

## Инструкция по установке

### 1) Клонируйте репозиторий:
```git clone https://github.com/AJLbN0H/diploma```
### 2) Создайте виртуальное окружение:
- Для Linux/MacOS:\
```python -m venv venv``` \
```source venv/bin/activate```
- Для Windows:\
```python -m venv venv``` \
```venv/Scripts/activate```
### 3) Установите зависимости:
```pip install -r requirements.txt```
### 4) Создайте базу данных через ```PostgresSQL```
### 5) Заполните своими данными файл ```.env.sample``` и переимнуйте его в ```.env```

## API endpoints
### Аутентификация:
- ```POST``` ```users/register/``` - Регистрация
- ```POST``` ```users/login/``` - Авторизация (JWT)
- ```POST``` ```users/token/refresh/``` - Оновление JWT токена
### Материалы:
- ```GET``` ```materials/material/``` - Получение списка материалов
- ```GET``` ```materials/material/{material_id}/``` - Получение подробной информации о материале
- ```POST``` ```materials/material/``` - Создание материала
- ```PUT/PATCH``` ```materials/material/{material_id}/``` - Обновление информации о материале
- ```DELETE``` ```materials/material/{material_id}/``` - Удаление материала
### Разделы:
- ```GET``` ```materials/section/``` - Получение списка разделов
- ```GET``` ```materials/section/{section_id}/``` - Получение подробной информации о разделе
- ```POST``` ```materials/section/``` - Создание раздела
- ```PUT/PATCH``` ```materials/section/{section_id}/``` - Обновление информации о разделе
- ```DELETE``` ```materials/section/{section_id}/``` - Удаление раздела
### Тесты:
- ```GET``` ```tests/test/``` - Получение списка тестов
- ```GET``` ```tests/test/{test_id}/``` - Получение подробной информации о тесте
- ```POST``` ```tests/test/``` - Создание теста
- ```PUT/PATCH``` ```tests/test/{test_id}/``` - Обновление информации о тесте
- ```DELETE``` ```tests/test/{test_id}/``` - Удаление теста
### Вопросы:
- ```GET``` ```tests/question/``` - Получение списка вопросов
- ```GET``` ```tests/question/{question_id}/``` - Получение подробной информации о вопросе
- ```POST``` ```tests/question/``` - Создание вопроса
- ```PUT/PATCH``` ```tests/question/{question_id}/``` - Обновление информации о вопросе
- ```DELETE``` ```tests/question/{question_id}/``` - Удаление вопроса
### Ответы:
- ```GET``` ```tests/answer/``` - Получение списка ответов
- ```GET``` ```tests/answer/{answer_id}/``` - Получение подробной информации о ответе
- ```POST``` ```tests/answer/``` - Создание ответа
- ```PUT/PATCH``` ```tests/answer/{answer_id}/``` - Обновление информации об ответе
- ```DELETE``` ```tests/answer/{answer_id}/``` - Удаление ответа
### Тестирование:
- ```GET``` ```tests/detail/{test_id}``` - Получение теста
- ```POST``` ```tests/submit/{test_id}/``` - Отправка ответов на вопросы теста
- ```GET``` ```tests/results/``` - Получение списка результатов тестов
- ```PUT/PATCH``` ```tests/results/{test_id}/detail/``` - Получение детально информации результата теста
- ```DELETE``` ```tests/results/{test_id}/delete/``` - Удаление результата теста

## Реализована система ролей и прав доступа: 

1. ### `Администраторы` 
- полный доступ ко всем функциям платформы.
2. ### `Владельцы курсов (Преподаватели)` 
- доступ к управлению своими курсами, материалами и тестами.
3. ### `Студенты` 
- доступ только к просмотру материалов и прохождению тестов.
